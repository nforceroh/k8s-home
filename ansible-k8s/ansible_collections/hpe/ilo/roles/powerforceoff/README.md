powerforceoff
=========

Force power off on all iLO systems.

Role Variables
--------------

Credentials of the iLO like `baseuri`, `username`, `password` needs to mentioned. 

Example Playbook
----------------

An example of how to use the role: 

    - hosts: servers
      roles:
         - powerforceoff
